Embedding a libGDX app on a web page

For this compendium to be as beautiful as it is, I wanted to be able to add interactive mini apps or demos alongside code snippets to the page so people would see the results of the guide, tutorial or documentation they're following. In order to achieve this I followed these steps:

  • Have the app published somewhere on the web
  • Use an iframe element to include the app on a page
